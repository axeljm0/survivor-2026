# Survivor 2026

**Survivor 2026** es un pequeño juego 2D estilo *survival* desarrollado con **Unity** y **C#** como proyecto académico.
El objetivo del juego es sobrevivir mientras aparecen enemigos continuamente, obteniendo experiencia y mejorando el nivel del jugador.

## Características

* Movimiento del jugador en entorno 2D
* Aparición automática de enemigos
* Sistema de experiencia y niveles
* Barra de vida y barra de experiencia
* Interfaz gráfica utilizando Unity UI
* Sistema básico de puntuación
* Uso de consultas con **LINQ**
* Integración con **SQLite** para manejo de datos

## Tecnologías utilizadas

* C#
* Unity
* SQLite
* LINQ
* Programación Orientada a Objetos (POO)

## Instalación

1. Clonar el repositorio:

```bash
git clone https://github.com/axeljm0/survivor-2026.git
```

2. Abrir el proyecto utilizando **Unity Hub**.

3. Ejecutar la escena principal del juego.

## Cómo jugar

* **W / A / S / D** → mover al personaje
* El jugador gana experiencia al derrotar enemigos
* El objetivo es sobrevivir el mayor tiempo posible

## Base de datos

El proyecto utiliza **SQLite** para almacenar información como puntuaciones u otros datos del juego.
La base de datos puede visualizarse con herramientas como SQLiteStudio o DB Browser for SQLite.
